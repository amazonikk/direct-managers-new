# Direct Managers Dashboard

Повний GitHub Pages дашборд для Таї та Катерини.

## Що виправлено в цій версії

- кнопки місяців формуються **глобально з обох Google Таблиць**;
- якщо серпень є лише в таблиці Таї, кнопка `Серпень 2026 р.` буде видима також у вкладці Катерини;
- додані швидкі кнопки `Таблиця Тая` і `Таблиця Катерина` біля вкладок direct-менеджерів;
- Google XLSX завантажується без кешу;
- парсер розширює діапазон аркуша, якщо Google залишив застарілий `!ref` і нові рядки опинилися нижче старої межі;
- GitHub Actions перевіряє таблиці кожні 5 хвилин;
- `data/report.json` автоматично комітиться у `main`, тому оновлення працює навіть коли Pages налаштовано як `Deploy from a branch → main → /(root)`;
- детальний перегляд тижня залишається доступним натисканням на картку тижня.

## Файли для завантаження в корінь репозиторію

```text
.github/workflows/deploy-pages.yml
data/report.json
scripts/build-data.mjs
.nojekyll
app.js
index.html
package.json
README.md
styles.css
```

Замініть весь вміст репозиторію файлами з цієї папки та зробіть один commit.

## Після завантаження

1. Відкрийте `Actions`.
2. Виберіть `Update Google Sheets data`.
3. Натисніть `Run workflow`.
4. Дочекайтеся зеленої галочки.
5. Оновіть сайт через `Ctrl + F5`.

## Доступ до Google Таблиць

Обидві таблиці повинні мати режим:

`Усі, хто має посилання → Читач`.

## Автоматичні нові місяці

Код не містить вручну прописаного списку місяців. Будь-яка коректна дата виду `3.08.2026`, `03.08.2026` або справжня дата Google Sheets перетворюється на `2026-08-03`. Після оновлення звіту місяць автоматично з’являється у верхньому перемикачі.


## Налаштування GitHub Pages

Для цієї версії залиште саме:

`Settings → Pages → Deploy from a branch → main → /(root)`

Workflow оновлює `data/report.json` у гілці `main`. Після його commit GitHub Pages автоматично перевидає сайт.

## Логіка тижнів

- Перший тиждень місяця: від 1 числа до найближчої неділі.
- Наступні тижні: понеділок–неділя.
- Останній тиждень завершується останнім днем місяця.

Наприклад, для липня 2026: 1–5, 6–12, 13–19, 20–26, 27–31.
