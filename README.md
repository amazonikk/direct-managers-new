# Direct Managers Dashboard — live Google Sheets

Ця версія читає дані безпосередньо з Google Таблиць у браузері. GitHub Actions не потрібен.

## Запуск

1. Завантажте весь вміст цієї папки в корінь репозиторію.
2. GitHub Pages: `Deploy from a branch` → `main` → `/(root)`.
3. Обидві Google Таблиці повинні мати доступ `Усі, хто має посилання → Читач`.
4. Після deploy відкрийте сторінку через Ctrl+F5.

## Оновлення

- Кнопка «Оновити зараз» реально повторно читає Google Sheets.
- Автоматичне оновлення виконується раз на 5 хвилин.

- `data/report.json` використовується лише як резервна копія, коли Google недоступний.
