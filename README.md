# Direct Managers Dashboard — Google Apps Script

Ця версія отримує дані з приватних Google-таблиць через Google Apps Script Web App.

## Публікація на GitHub Pages

1. Завантажте всі файли цієї папки в корінь репозиторію.
2. У `Settings → Pages` виберіть `Deploy from a branch`, `main`, `/(root)`.
3. Після deployment відкрийте сайт і натисніть «Оновити зараз».

Apps Script URL вже записаний у `app.js`.
